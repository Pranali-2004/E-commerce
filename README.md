# Project Description

Project Name: -E-commerce Website

Project Overview: -
This project is a fully functional e-commerce website built using React. It demonstrates an understanding of React fundamentals such as routing, state management, API integration, and the Context API. This is Single Page Application(SPA) that provides fast and smooth navigation without page reloads.

Project Objectives: -
Create user friendly E-Commerce platform for browsing and purchasing products using React.
Ensure responsiveness for mobile, tablet and desktop devices.
Implement secure transactions and well structured shopping cart.
To display products dynamically with details using API based data fetching.
To improve user experience using fast performance and easy navigations.

Scope of the project: -
Products with categories
Shopping Cart with Checkout

Technologies Used: -
HTML, CSS, and React.

# How to run the project?

1. Make sure to install Node.js
2. Open the folder in the terminal
3. Install Dependencies using: npm install
4. Run the project using: npm run dev
5. Click on the link shown in the terminal like: (http://localhost:5173)

# Features implemented: -

1. Components: -
   Components are the building blocks of a React application. Components helps to keep the reusable and organized.

2. State Management (useState and useEffect): -
   Used to changes the User Interface based on the data and user actions.

   useState: -helps to store and update values like cart items.

   useEffect : -used to perform API data fetching.

3. React Router: -
   Used for navigations in the application. Allows users to move between pages like Home, Products, Cart and Checkout without reloading the page.

4. Context API: -
   Context API s used to share data(state, functions) across multiple components without passing props again and again. In this project, it is used to manage cart data globally.

5. API Data Fetching: -
   Used to get product data from an external source (like FakeStore API). Using this the data is displayed on the User Interface.

6. Conditional Rendering: -
   Conditional rendering means showing different UI elements based on conditions. In this project, it is used to showing loader while data is loading or dispaying a message when cart is empty.
